ZCAP is the treasury-backed membership token of Zion Capital, built on Solana. This repository stores the public metadata, token assets, treasury policy documents, branding files, and related resources used across wallets, explorers, DEXs, aggregators, and blockchain indexers.

## About ZCAP

ZCAP is designed as a treasury-backed membership token that combines:

* Long-term treasury growth
* Community participation
* Governance utility
* Premium membership access
* Transparent reserve policies
* Disciplined capital management

Unlike speculative meme tokens, ZCAP is structured around treasury accumulation, reserve backing, and long-term ecosystem value.

## Treasury Philosophy

Zion Capital is built around the principle that token ecosystems should be transparent, disciplined, and backed by real reserves.

ZCAP is intended to serve as a treasury-backed membership token where value is supported by:

* Treasury reserve accumulation
* Controlled token supply management
* Transparent public disclosures
* Sustainable liquidity practices
* Long-term membership utility

The treasury strategy emphasizes capital preservation, responsible deployment, and gradual growth rather than short-term speculation.
